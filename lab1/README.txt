Звіт по Python-коду

Код містить три функції: task1, task2, та task3, які виконують окремі завдання: обчислення довжини рядка,
виконання арифметичних операцій та пошук максимального значення у списку.

def task1(str):
    return len(str)

    Призначення: Обчислює довжину переданого рядка.
    Вхідні дані: Один параметр str - рядок (string).
    Вихідні дані: Довжина рядка як ціле число (int).

Докладний опис:
Функція task1 отримує рядок і повертає його довжину за допомогою вбудованої функції len(). Це корисно для визначення кількості символів у рядку, включаючи пробіли та спеціальні символи.

Приклад використання:

print(task1("hello"))  # Виведе: 5

 Функція task2

    Призначення: Виконує арифметичну операцію між двома числами на основі переданого оператора.
    Вхідні дані: Три параметри:
        a (число, тип int або float) - перший операнд.
        operator (рядок, тип str) - оператор арифметичної операції ("+", "-", "/", "//", "**", "*")
        b (число, тип int або float) - другий операнд.
    Вихідні дані: Результат операції між a та b (тип int або float залежно від операції).

Докладний опис:
Функція task2 отримує два числа і рядок-оператор, який визначає арифметичну операцію, що має бути виконана: додавання (+), віднімання (-), ділення (/), цілочисельне ділення (//), піднесення до степеня (**) або множення (*). Функція використовує умовні оператори if-elif для вибору відповідної операції.

Приклад використання:

print(task2(4, "+", 2))  # Виведе: 6
print(task2(4, "**", 2)) # Виведе: 16

Функція task3
def task3(list):
    return max(list)

    Призначення: Повертає максимальне значення у списку чисел.
    Вхідні дані: Один параметр list - список чисел (list типу int або float).
    Вихідні дані: Максимальне значення у списку (тип int або float).

Докладний опис:
Функція task3 приймає список чисел і повертає найбільше значення за допомогою вбудованої функції max(). Це корисно для пошуку найбільшого елемента в наборі числових даних.

Приклад використання:

print(task3([1, 5, 3]))  # Виведе: 5
