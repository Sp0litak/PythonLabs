Task 1: Clean Data

    Функція clean_data очищує рядок даних, розділений комами, від пробілів і переводить всі символи в нижній регістр.
    Використовані функції split(',') для розділення рядка на частини по комі, strip() для видалення зайвих пробілів по боках кожного елемента, і lower() для переведення в нижній регістр.

Task 2: Filter Emails

    Функція filter_emails фільтрує список електронних адрес за наявністю одного символу '@' у кожному з них.
    Використовано вбудовані функції filter() і map() для виконання цієї задачі.

Task 3: Extract Keywords

    Функція extract_keywords витягує слова з рядка, які є довшими за задану довжину.
    Використано функції filter() і map() для обробки рядка та відбору слів за заданим критерієм.

Task 4: Process Text Data

    Функція process_text обробляє текст, видаляючи спеціальні символи, переводить його в нижній регістр і відфільтровує короткі рядки.
    Використано модуль re для роботи з регулярними виразами, map() і filter() для обробки та фільтрації даних.

Task 5: Normalize Data

    Функція normalize_data нормалізує числові дані до діапазону від 0 до 1 на основі максимального значення.
    Використовані функції map() для перетворення рядка на список чисел і подальше нормалізація значень.

Task 6: Concatenate Strings

    Функція concatenate_strings об'єднує рядки, розділені спеціальним роздільником, в один рядок без цього роздільника.
    Використовано метод split() для розділення рядка і join() для об'єднання його частин.

Task 7: Sum Numeric Strings

    Функція sum_numeric_strings обчислює суму числових значень, розділених комами, у рядку.
    Використано функції filter() і map() для відбору числових значень та їх перетворення в числа для подальшого обчислення суми.

Task 8: Filter Numbers

    Функція filter_numbers відфільтровує числа у рядку, що перевищують заданий поріг.
    Використано вбудовані функції filter() і float() для фільтрації і перевірки числових значень.

Task 9: Map to Squares

    Функція map_to_squares обчислює квадрати чисел, розділених комами, у рядку.
    Використовано функцію map() для перетворення кожного числа на його квадрат.

Task 10: Reverse Strings

    Функція reverse_strings обертає кожне слово у рядку задом наперед.
    Використано функцію map() і зріз [::-1] для обертання кожного слова у рядку.
