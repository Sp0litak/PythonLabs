Звіт по Python-коду

Код містить три функції: task1, task2, та task3, які виконують порівняння чисел, обчислення максимальної довжини тексту або числа, а також перевірку рівності трьох чисел.
1. Функція task1
    Призначення: Виконує порівняння двох чисел на основі переданого оператора порівняння.
    Вхідні дані:
        number1 (число, тип int або float): Перше число.
        operator (рядок, тип str): Оператор порівняння (">", "<", ">=", "<=", "==", "!=").
        number2 (число, тип int або float): Друге число.
    Вихідні дані: Результат порівняння як логічне значення (bool).

Докладний опис:
Функція task1 отримує два числа та оператор порівняння у вигляді рядка. На основі оператора функція порівнює number1 з number2 і повертає результат у вигляді True або False. Використовує умовні оператори if-elif для вибору відповідного порівняння.

Приклад використання:

print(task1(4, ">", 2))  # Виведе: True
print(task1(4, "==", 4)) # Виведе: True

2. Функція task2
    Призначення: Повертає максимальне значення між довжиною рядка та числом.
    Вхідні дані:
        text (тип str): Рядок.
        number (тип int): Число.
    Вихідні дані: Максимальне значення між довжиною рядка та числом (тип int).

Докладний опис:
Функція task2 отримує рядок text та число number. Вона обчислює довжину рядка за допомогою функції len(), а потім повертає максимальне значення між довжиною рядка та числом, використовуючи функцію max().

Приклад використання:

print(task2("hello", 10))  # Виведе: 10
print(task2("hello", 3))   # Виведе: 5

3. Функція task3
    Призначення: Перевіряє, чи є три числа рівними.
    Вхідні дані:
        number1 (число, тип int або float): Перше число.
        number2 (число, тип int або float): Друге число.
        number3 (число, тип int або float): Третє число.
    Вихідні дані: Логічне значення (bool), яке вказує, чи всі три числа рівні.

Докладний опис:
Функція task3 приймає три числа і перевіряє їх на рівність. Вона використовує ланцюжок операторів ==, що дозволяє перевірити рівність всіх трьох чисел одночасно. Функція повертає True, якщо всі три числа рівні, і False в іншому випадку.

Приклад використання:

print(task3(5, 5, 5))  # Виведе: True
print(task3(5, 6, 5))  # Виведе: False
