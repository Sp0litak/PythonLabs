1. Функція task1
    Призначення: Перетворює введене значення віку у ціле число.
    Вхідні дані:
        age (тип str): Рядок, що містить вік.
    Вихідні дані:
        Якщо успішно, повертає ціле число age.
        Якщо age не може бути перетворено у ціле число (ValueError), повертає помилкове повідомлення.

Детальний опис:
Функція спробує перетворити age у ціле число за допомогою int(). Якщо операція успішна, поверне отримане ціле число. Якщо age не є допустимим числовим рядком (викликає ValueError), поверне повідомлення про помилку з проханням ввести числове значення.

Приклад використання:

result = task1("25")
print(result)  # Виведе: 25

result = task1("двадцять п'ять")
print(result)  # Виведе: "Помилка: Будь ласка, введіть числове значення для віку."

2. Функція task2

    Призначення: Обчислює добуток двох чисел.
    Вхідні дані:
        num1, num2 (тип str): Рядки, що містять числа для множення.
    Вихідні дані:
        Якщо успішно, повертає добуток num1 і num2.
        Якщо одне з чисел не може бути перетворено у ціле число (ValueError), повертає помилкове повідомлення.

Детальний опис:
Функція спробує перетворити num1 і num2 у цілі числа за допомогою int(), обчислить їх добуток і поверне результат. Якщо одне з чисел не є допустимим числовим рядком (викликає ValueError), поверне помилкове повідомлення з проханням ввести числові значення.

Приклад використання:

result = task2("5", "3")
print(result)  # Виведе: 15

result = task2("п'ять", "3")
print(result)  # Виведе: "Помилка: Будь ласка, введіть числові значення для чисел."

3. Функція task3

    Призначення: Обчислює довжину рядка.
    Вхідні дані:
        input_str (тип str): Рядок для обчислення довжини.
    Вихідні дані:
        Якщо input_str є рядком, повертає його довжину.
        Якщо input_str не є рядком, повертає помилкове повідомлення.

Детальний опис:
Функція спочатку перевіряє, чи є input_str рядком за допомогою isinstance(). Якщо так, обчислює і повертає його довжину за допомогою len(). Якщо input_str не є рядком, повертає помилкове повідомлення з проханням ввести рядок, а не числове значення.

Приклад використання:

result = task3("Привіт, світ!")
print(result)  # Виведе: 13

result = task3(25)
print(result)  # Виведе: "Помилка: Будь ласка, введіть рядок, а не числове значення."

4. Функція task4

    Призначення: Обчислює суму всіх цілих чисел у списку.
    Вхідні дані:
        int_list (тип list): Список цілих чисел для обчислення суми.
    Вихідні дані:
        Якщо усі елементи int_list є цілими числами, повертає їхню суму.
        Якщо int_list містить не цілі числа або інші типи даних, повертає None.

Детальний опис:
Функція спочатку перевіряє за допомогою all() чи всі елементи int_list є цілими числами (isinstance(x, int)). Якщо це так, обчислює суму цих чисел за допомогою sum(). Якщо int_list містить не цілі числа або інші типи даних (викликає TypeError), повертає None.

Приклад використання:

result = task4([1, 2, 3, 4])
print(result)  # Виведе: 10

result = task4([1, 2, "три", 4])
print(result)  # Виведе: None

5. Функція task5

    Призначення: Обчислює середній бал для кожного студента у списку кортежів (ім'я, оцінки).
    Вхідні дані:
        data (тип list): Список кортежів (ім'я, оцінки), де оцінки є списком чисел.
    Вихідні дані:
        Якщо обробка успішна, повертає список кортежів (середній бал, ім'я).
        Якщо виникає будь-яка помилка під час обробки, повертає повідомлення про помилку.

Детальний опис:
Функція ітерується по data та обчислює середній бал для кожного студента, використовуючи sum() та len() для оцінок. Результати записуються у вихідний список як кортежі (середній бал, ім'я). Якщо виникає будь-яка помилка під час обробки (Exception), функція повертає повідомлення про помилку.

Приклад використання:

data = [("Олег", [85, 90, 92]), ("Ірина", [78, 80, 85])]
result = task5(data)
print(result)  # Виведе: [(89.0, "Олег"), (81.0, "Ірина")]

data = [("Олег", [85, 90, "вісімдесят"]), ("Ірина", [78, 80, 85])]
result = task5(data)
print(result)  # Виведе: "Помилка обробки списку!"
