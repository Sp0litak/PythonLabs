1. Функція task1Призначення: 
Перевіряє, чи складається рядок тільки з малих літер та цифр.
Вхідні дані:

    input_str (тип str): Рядок для перевірки.

Вихідні дані:

    True, якщо рядок складається тільки з малих літер та цифр.
    False, якщо рядок містить будь-які інші символи або великі літери.

    input_str (тип str): Рядок для перевірки.

Вихідні дані:

    True, якщо рядок містить хоча б одну велику літеру.
    False, якщо рядок не містить жодної великої літери.
